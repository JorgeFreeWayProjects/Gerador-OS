Gerador de OS

Instruções de Uso:

1. Quando você gerar texto com ID OS em branco, vai gerar modelos de OS para criar atendimento.
2. Quando abrir atendimento no IXC, vai gerar a OS com ID, então você insere no campo ID OS e gera novamente uma OS para enviar no WhatsApp.
3. Não esqueça de sempre confirmar Contato e Endereço com cliente.
4. Use "Limpar Campos" para resetar todos os campos e iniciar uma nova OS.
5. Colaborador, Tipo OS, Nome do Cliente, periodo e data são obrigatórios!!!
